# ClearView Dashboard

ClearView is a sample secure mission dashboard demonstrating UI/UX design and full-stack development skills. This repository contains the scaffolding for a React + TypeScript frontend and an Express backend with mocked MongoDB data.

## Features and Tech Stack

- **Frontend:** React, TypeScript, Redux Toolkit (placeholder)
- **Backend:** Node.js, Express, mocked MongoDB
- **DevOps:** Docker Compose, GitHub Actions (placeholder)

## Setup Instructions

1. Install dependencies (requires Node.js):
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Start the API server:
   ```bash
   node server/index.js
   ```

## What This Project Demonstrates

This project showcases the initial setup for a secure dashboard, including a modern frontend stack and a simple backend API. Further phases will expand on functionality, tests, and CI/CD.
